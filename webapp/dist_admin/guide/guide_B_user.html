FORMS USER GUIDE!!

너비

너비는 요소의 너비를 변경 할 수 있습니다.

너비 값은 10px 부터 500px 까지 제공되며, 단위는 10px 입니다. (10px, 20px, 30px...., 500px)

기본 너비

px 너비

너비는 class 추가로 설정되며, 너비 class를 중복제공 할 수 없습니다.

ex) class="w100"

% 너비

너비는 class 추가로 설정되며 ( 5%, 10%, 15%, 20%, 25%, 33%, 50%, 100% )가 제공됩니다.

ex) class="w25p"

간격(바깥쪽)

간격은 요소의 안쪽(padding)과 바깥쪽(margin)의 간격을 변경 할 수 있습니다.

간격은 5px 부터 70px 까지 제공되며, 단위는 5px 입니다. ( 5px, 10px, 15px, 20px,...... 70px )

기본

바깥쪽(margin) 간격 - 단일

간격은 class 추가로 설정 됩니다.

ex) class="mr20"

바깥쪽(margin) 간격 - 중복

간격은 class 중복 제공도 가능합니다.

ex) class="mr20 ml20"

간격(안쪽)

간격은 요소의 안쪽(padding)과 바깥쪽(margin)의 간격을 변경 할 수 있습니다.

간격은 5px 부터 70px 까지 제공되며, 단위는 5px 입니다. ( 5px, 10px, 15px, 20px,...... 70px )

기본

안쪽(padding) 간격 - 단일

간격은 class 추가로 설정 됩니다.

ex) class="pl10"

안쪽(padding) 간격 - 중복

간격은 class 중복 제공도 가능합니다.

ex) class="pr10 pl10"

빈 박스

그냥 빈박스 입니다.

ex) class="vacantBox"

빈박스

요소의 위치

해당 박스 안에 있는 요소는 값에 따라 위치를 변경 할 수 있습니다.

위치는 class로 설정하고 ( 좌, 우, 중앙1, 중앙2)를 제공합니다.

위치1 (좌측)

ex) class="textLeft"

위치2 (중앙-좌우)

ex) class="textCenter"

위치3 (우측)

ex) class="textRight"

위치4 (중앙-상하)

상하로 정렬되지 않는 요소들을 중앙 정렬 시켜 줍니다.

ex) class="textMiddle"
